A Kenya-first blockchain marketplace connecting verified farmers with trusted buyers through transparent pricing, fair markets, and guaranteed sales. Where verified farmers meet trusted buyers with end-to-end visibility.

Key Features:

  • Blockchain-powered traceability for all crops
  • Real-time connection between farmers and buyers
  • Verified farmer onboarding with KYC compliance

AI & Agile Tools

YodaAI

YodaAI is an intelligent, AI-powered retrospective assistant designed to help agile teams run engaging and structured sprint reviews using the 4Ls framework (Liked, Learned, Lacked, and Longed For). It transforms traditional retrospectives into dynamic, conversation-driven sessions guided by AI.

Key Features:

  • AI-guided 4Ls framework facilitation
  • Automatic theme identification and grouping
  • Team voting system for priority themes
  • Disciplined Agile-based recommendations
  • Comprehensive PDF report generation
  • Action items tracking and follow-up
